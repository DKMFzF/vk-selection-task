# VK Selection Task

Приложение клиента фильмов сделано в рамках отбора на стажировку VK

## Стек

- React
- TypeScript
- React Router
- Effector + effector-react
- VKUI
- Axios

## Start

```bash
git clone https://github.com/DKMFzF/vk-selection-task.git
cd vk-selection-task
npm i
# создать файл .env и добавить KINOPOISK_API_KEY
npm run dev
```

## .env

`.env.default`:
- `PORT` - порт dev-сервера
- `KINOPOISK_API_URL` - base url API
- `KINOPOISK_API_KEY` - API ключ kinopoisk.dev
